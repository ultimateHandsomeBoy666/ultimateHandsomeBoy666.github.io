🏠 关于本站

这是一个用 Gridea 搭建的静态博客站点，主要是放我自己写的一些技术博客。

做 android 开发也有一年多了，但是写的文章其实不算很多，一方面是工作的时候可能就没啥时间，另一方面也怪自己懒，有些技术上的点自己记录下来，但是却迟迟不整理成文章，不做输出，😓。这样造成了两方面的后果，一是没有输出，就没有系统的总结和思考；二是比较现实的一点，找工作的时候简历上本来可以写上博客地址的，结果写不了，白白丢分。

所以从去年底开始就想搭建一个自己的博客，然而能加快输出的频率。当然，我不想写一些大家都写过的东西，反反复复地讲，也挺没意思的。我想写一些自己的总结，并且一定要有自己的思考，还要尽量通俗易懂。因为没有买服务器，所以直接就用了 github pages 来搭建静态的站点，然后在网上找到了 Gridea 这个客户端，就尝试起来，搭建自己的博客（BTW，Gridea 有些小 bug 很蛋疼🤦‍♀️）。

写下这段文字的时候是 2021 年 3 月，现在我才刚刚搭建起这个博客。以后会努力多进行输出，✌️

👨‍💻 博主是谁

可以叫我牛蛙。

虽然现在做着3年以后就要消失的 android 客户端，但我感觉还是挺开心的，不像很多人说的那么悲观焦虑，这也与我的经历过关吧。我本身是非科班出身的，大学学的是化工专业。所谓材化环生，四大天坑，我能从四大天坑中跳出来，还是挺好的，这不比做化工强多啦。

而且我大学时本身也对编程挺感兴趣的，大学时学的 C 语言课拿了 90多分，而我的同学们学起 C 语言大都苦不堪言。大三的时候自己照着 BigNerdRanch 那本书自学了 android 开发。至于为啥学客户端开发，这就是非科班的普遍现象吧，毕竟没有经过系统的科班训练，非科班对于编程的认识和兴趣都是来自于自己使用的客户端软件。

在那之后，人生经历了一些坎坷，人也懒散了许多，这里按下不表。不过从 19 年开始，算是正式进入到这个行业了。
